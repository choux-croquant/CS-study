### 3강 역, 이, 대우
- 조건 명제에서 사용되는 개념으로, 하나의 명제를 변형하는 방식.
- 증명하기 어려운 명제를 대우를 통해 증명하는 등의 방식으로 사용된다.
- p -> q의 관계에서
  - 역 : p와 q의 위치를 바꾸는 것
  - 이 : p와 q에 부정을 붙여 명제를 다르게 하는 것
  - 대우 : 역과 이를 모두 적용하는 것, 대우의 경우 원래의 조건 명제와 진리표가 동일하다.
